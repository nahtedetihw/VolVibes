## VolVibes


## Add vibrations to the volume buttons!!
* ONLY COMPATIBLE WITH IOS 13 & 14
* Control the volume stepper amount!
* Hide the volume HUD!
* Change HUD Color!
* Remove Speaker Glyph!
* Add Volume Percent!
* Change HUD style!
